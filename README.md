🚀 FounderLens AI
An AI-powered Decision Support System for Startup Founders

📌 Overview
FounderLens AI is an advanced, full-stack decision support system engineered to bridge the gap between initial startup ideation and market reality. Built for early-stage founders, the platform utilizes Natural Language Processing (NLP) and high-speed machine learning inference to dissect business pitches and objectively evaluate problem-solution fit.

⚙️ Tech Stack
Frontend: React.js, GSAP, Web Audio API

Backend: Python (FastAPI / Flask)

AI Integration: NLP, Groq, Machine Learning models

Database: SQL for tracking user telemetry logs and pitch scores
